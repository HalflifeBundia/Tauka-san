# 踏歌サン

一个以`三角波`作为元音材料制作的无生物音源，适用于[OpenUtau](https://github.com/stakira/OpenUtau)
> *目前为预览版，正式版中会包含立绘*

## 音源简介

**中文名:** `踏歌叁`  
**日文名:** `踏歌サン`  *读音：Tauka-san*  
**舷号:** `103`  
**语言:** `日本語`  
**类型:** `CV`,`平假名标注`  
**编码:** `GB2312`,`Shift JIS`  
**音域:** `C3-A5`  
> *收录音素不包括**语尾息**【a R...】,**母音+无声子音**【a k...】,**呼吸音**【br1...】*

推荐使用`JA VCV&CVVC`音素器  
推荐使用`worldline-R`,`moresampler`,`hifisampler`重采样器
